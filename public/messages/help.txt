> `help` - prints help.

>`projects` - prints all available projects projects.

> To update or create entry at specific date, write `update DAY.MONTH.YEAR PROJECT_NAME HOURS:MINUTESCOMMENT(OPTIONAL)`.
> For example, `update 25.07.2016 fame and partners 8:00 a nice comment`.

> To enter time when I ask you, write `PROJECT_NAME HOURS:MINUTES COMMENT(OPTIONAL)`.
> For example, `fame and partners 8:00 some comment`.

> To add new project, write `add project PROJECT_NAME`.
> For example, `add project Fame and partners`.

> To get report for a week, write `show week`.

> To get report since the beginning of the month, write `show month`.

> If you are absent on some days, write `set REASON DAY.MONTH.YEAR - DAY.MONTH.YEAR`.
> For example, `set vacation 15.08.2016 - 19.08.16`.
> Valid reasons: Vacation, Illness.

> If you want bot to ask you, write `ask me`.
